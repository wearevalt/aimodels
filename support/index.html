Questions fréquentes
Comment soumettre un modèle IA ?
Rendez-vous sur la page Soumettre une IA. Remplissez le formulaire avec le nom, l'URL officielle et une description du modèle. Notre équipe valide chaque soumission sous 48h.
Comment créer un compte ?
Cliquez sur "Connexion" en haut à droite, puis sur "Créer un compte". Vous pouvez vous inscrire avec votre email ou via Google/GitHub.
Comment accéder à l'API Valtlens ?
L'API est disponible sur la page API. Créez un compte, générez une clé API dans vos paramètres, et consultez la documentation pour démarrer.
Comment me désabonner de la newsletter ?
Cliquez sur le lien "Se désabonner" en bas de chaque email que nous vous envoyons. Votre désabonnement est effectif immédiatement.
Un modèle que j'ai soumis n'est pas encore visible, pourquoi ?
Chaque soumission est vérifiée manuellement par notre équipe. Le délai est de 24 à 48h. Si votre soumission n'apparaît pas après 48h, contactez-nous à support@valtlens.com.
Comment signaler une information incorrecte ?
Envoyez-nous un email à hello@valtlens.com en précisant le nom du modèle et l'information incorrecte. Nous ferons la correction dans les 24h.
