# AI Video Models Directory

A comprehensive directory website showcasing the best AI video generation models available in 2026.

## 🎬 Overview

This project is a modern, responsive web directory that provides detailed information about the top AI video generation models. It features:

- **Featured Models**: Highlighted top picks with detailed descriptions
- **Complete Model Listings**: Comprehensive database of 15+ AI video models
- **Advanced Filtering**: Filter by text-to-video, image-to-video, free tier, and API availability
- **Search Functionality**: Quick search across model names, developers, and features
- **Comparison Table**: Side-by-side comparison of key model specifications
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices

## 🚀 Features

### Model Information
Each model card includes:
- Model name and developer
- Detailed description
- Maximum video duration
- Resolution capabilities
- Pricing model
- Best use cases
- Feature tags
- Star ratings
- Direct links to official websites

### Interactive Features
- **Real-time Search**: Press `/` to focus search, `Escape` to clear
- **Category Filters**: Filter models by type and features
- **Smooth Scrolling**: Navigate between sections effortlessly
- **Hover Animations**: Engaging visual feedback on interactions
- **Responsive Grid**: Adaptive layout for all screen sizes

## 🚀 Getting Started

### Viewing the Website

1. Navigate to the project directory:
   ```bash
   cd ai-video-models-directory
   ```

2. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local development server:
     ```bash
     # Using Python
     python -m http.server 8000
     
     # Using Node.js (if you have http-server installed)
     npx http-server
     ```

3. The website will open in your default browser

### No Build Process Required

This is a static website that requires no build process or dependencies. Simply open the HTML file in a browser to view it.

## 🔧 Customization

### Adding New Models

To add a new model, edit the `modelsData` array in `script.js`:

```javascript
{
    id: 16,
    name: "Model Name",
    developer: "Developer Name",
    description: "Model description...",
    maxDuration: "10 seconds",
    resolution: "1080p",
    pricing: "Freemium",
    bestFor: "Use case",
    rating: 4.5,
    tags: ["Tag1", "Tag2", "Tag3"],
    categories: ["text-to-video", "free"],
    url: "https://example.com"
}
```

### Modifying Styles

Edit `styles.css` to customize:
- Colors: Modify CSS variables in `:root`
- Fonts: Change the Google Fonts import and font-family
- Layout: Adjust grid and flexbox properties
- Animations: Modify keyframes and transitions

### Updating Content

Edit `index.html` to:
- Change the header and tagline
- Update featured models
- Modify the comparison table
- Edit the about section
